# Lucky Engine Dashboard - Dependencies

To initialize and run this project, ensure you have Node.js (>= 18) installed, and install the following dependencies.

## Core Installation Command
Run the following in the root of the `unified-dashboard` project:

```bash
npm install axios zustand lucide-react uuid
npm install -D @types/uuid
```

*(Note: `tailwindcss`, `postcss`, `autoprefixer`, `@heroicons/react` and standard Next.js dependencies were installed during the initial `create-next-app` initialization or earlier phases).*
